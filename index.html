안녕하세요? Seolmango입니다!

2022/07/10 by Seolmango

안녕하세요!! 어떤 방법으로 이곳을 찾았는지는 모르겠지만 오셨군요... 아주 원시적인 방법으로 구현된 웹페이지?블로그?입니다!

이곳은 저 Seolmango가 프로그래밍 연습을 핑계로 Github Pages를 이용해서 원시적으로 구현하고 있는 웹페이지 겸 블로그 입니다.

공부하면서 만들다보니 버그가 존재할 확률이 큽니다. 또한 바쁜 일상을 보내는 주인장의 특성상 활동이 적을 확률이 큽니다. 더 말하자면, 아무생각없이 글을 쓰기 때문에 맞춤법 오류가 있을 확률은 100% 입니다.

일일히 html 노가다를 통해 구현하는 블로그이다보니 댓글이나 반응 같이 이 페이지를 방문하는 여러분들과의 소통은 사실상 불가능합니다.
